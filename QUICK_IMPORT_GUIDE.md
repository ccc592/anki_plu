# 快速导入模式使用指南

## 🎯 功能概述

快速导入模式是v0.3.0新增的轻量级导入方式，通过小窗口弹窗完成快速导入，无需打开完整编辑器。

### 两种模式对比

| 特性 | 快速导入模式 | 完整编辑模式 |
|------|------------|------------|
| 触发方式 | Alt+A（默认） | 快速模式点击"📝 Preview" |
| 界面大小 | 350×450px 小窗口 | 全屏界面 |
| 位置 | 屏幕右下角 | 全屏 |
| 可拖动 | ✅ 是 | ❌ 否 |
| 使用场景 | 80%简单导入 | 20%需要编辑 |
| 平均耗时 | 2-3秒 | 15-20秒 |
| 背景遮罩 | 半透明 | 不透明 |

## 🚀 快速导入模式

### 基本使用

1. **选中文本**：包含"问："和"答："格式的内容
2. **按 Alt+A**：弹出快速导入窗口
3. **选择配置**：
   - Deck（卡组）- 自动记住上次选择
   - Model（模型）- 自动记住上次选择
   - Tags（标签）- 可选，逗号分隔
4. **快速导入**：
   - 点击"✓ Import"按钮，或
   - 按 Ctrl+Enter 快捷键
5. **等待反馈**：
   - 成功：绿色Toast提示 + 自动关闭
   - 失败：红色Toast提示 + 显示错误信息


### 特性说明

#### 1. 可拖动窗口
- **拖动方式**：鼠标按住标题栏拖动
- **位置记忆**：窗口位置自动保存到localStorage
- **下次打开**：在上次位置显示
- **边界限制**：不会拖出屏幕外

#### 2. 智能记忆
- **记住卡组**：最近5个卡组
- **记住模型**：最近3个模型
- **记住标签**：最近5组标签
- **自动填充**：下次打开自动填充上次选择

#### 3. Toast通知
- **成功导入**：绿色Toast，2秒自动消失
- **导入失败**：红色Toast，4秒自动消失，显示错误详情
- **非阻塞**：不影响继续操作

#### 4. 快捷键
- **Ctrl+Enter**：快速导入
- **ESC**：关闭窗口
- **点击背景**：关闭窗口

### 状态提示

**正常流程**：
```
1. "📦 识别到 X 张卡片" - 初始状态
2. "Importing..." - 导入中
3. "✓ Success!" - 导入成功（窗口0.8秒后自动关闭）
```

**错误流程**：
```
1. "📦 识别到 X 张卡片" - 初始状态
2. "Importing..." - 导入中
3. "Error: connection failed" - 显示错误（窗口保持打开）
4. Toast通知显示详细错误信息
```

## 📝 完整编辑模式

### 何时使用

- 需要查看所有卡片内容
- 需要修改卡片的问答内容
- 需要合并/拆分卡片
- 需要批量删除某些卡片
- 需要使用高级功能

### 切换方式

1. **从快速导入切换**：
   - 点击"📝 Preview"按钮
   - 切换到全屏编辑器
   
2. **直接打开**（未来版本）：
   - Alt+Shift+A（计划中）

### 完整编辑功能

- ✅ 左侧卡片列表预览
- ✅ 右侧卡片编辑器
- ✅ 批量操作工具栏（全选/删除）
- ✅ 合并/拆分卡片
- ✅ 6个键盘快捷键
- ✅ 智能选择器（搜索+置顶）

## 🐛 故障排查

### 快速导入窗口不显示

**可能原因**：
1. Shadow DOM未正确加载
2. 构建产物缺失

**解决方法**：
```bash
cd extension
npm run build
# 重新加载扩展
```

### 无法拖动窗口

**可能原因**：
1. 没有点击标题栏区域
2. 窗口已经在边界

**解决方法**：
- 确保鼠标按在"✓ Anki Quick Import"标题区域
- 检查窗口是否已贴边

### Toast通知不显示

**可能原因**：
1. ToastManager未正确初始化
2. Z-index被其他元素覆盖

**解决方法**：
- 检查控制台错误
- 确保z-index: 2147483647是最高

### 位置记忆失效

**可能原因**：
1. localStorage被清空
2. storageKey不匹配

**解决方法**：
```javascript
// 清空已保存位置
localStorage.removeItem('anki-assistant:popup-position:quick-import');
```

## 📊 性能对比

### 操作步骤对比

**快速导入（3步）**：
```
1. Alt+A → 弹窗
2. [可选]修改配置
3. Ctrl+Enter → 完成
耗时：2-3秒
```

**完整编辑（7步）**：
```
1. Alt+A → 全屏
2. 查看列表
3. 浏览内容
4. 选择卡组
5. 选择模型
6. 输入标签
7. 点击导入 → 完成
耗时：15-20秒
```

### 效率提升

- ⚡ **时间节省**：83-85%
- 🖱️ **点击减少**：约70%（7步 → 2步）
- 🔄 **上下文切换**：减少100%（不离开页面）
