# v0.3.0 快速导入模式测试清单

## 🎯 测试前准备

### 1. 构建和加载扩展
```bash
cd /home/100_Project/anki_plu/extension
npm run build
```

### 2. 加载到Chrome
1. 打开 `chrome://extensions/`
2. 启用"开发者模式"（右上角）
3. 点击"加载已解压的扩展程序"
4. 选择 `/home/100_Project/anki_plu/extension/dist`

### 3. 启动Anki
- 确保Anki正在运行
- 确保AnkiConnect插件已安装
- 打开任意卡组以激活Anki

---

## ✅ 核心功能测试

### Test 1: 快速导入模式 - 基本流程
**步骤**:
1. 打开任意网页（例如：https://example.com）
2. 复制以下测试内容到页面：
```
问：什么是JavaScript？
答：JavaScript是一种高级编程语言。

问：什么是React？
答：React是一个用于构建用户界面的JavaScript库。
```
3. 选中整段文本
4. 按 `Alt+A`

**预期结果**:
- ✅ 右下角弹出350×450px小窗口
- ✅ 显示"📦 识别到 2 张卡片"
- ✅ 卡组下拉框显示可用卡组
- ✅ 模型下拉框显示可用模型
- ✅ Tags输入框为空
- ✅ 显示"📝 Preview"和"✓ Import(2张)"按钮

---

### Test 2: 快速导入 - Ctrl+Enter快捷键
**步骤**:
1. 继续Test 1的状态
2. 选择一个卡组和模型
3. 按 `Ctrl+Enter`

**预期结果**:
- ✅ 显示"Importing..."状态
- ✅ 弹出绿色Toast："Successfully imported 2 cards!"
- ✅ Toast 2秒后自动消失
- ✅ 窗口0.8秒后自动关闭
- ✅ 打开Anki，确认卡片已导入

---

### Test 3: 窗口拖动功能
**步骤**:
1. 重新触发 `Alt+A` 打开快速导入窗口
2. 鼠标按住标题栏"✓ Anki Quick Import"
3. 拖动到屏幕左上角
4. 释放鼠标
5. 关闭窗口（ESC或点击✕）
6. 再次 `Alt+A`

**预期结果**:
- ✅ 窗口可以拖动
- ✅ 鼠标按下时光标变为"grabbing"
- ✅ 窗口不会拖出屏幕外
- ✅ 重新打开时，窗口在上次位置（左上角）

---
